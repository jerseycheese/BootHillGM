# Session Workflow

## Setup
1. Load context files
2. Set session goals
3. Start interaction log

## Management
1. Track context updates
2. Log decisions
3. Save artifacts

## Example
```markdown
Context: Boot Hill combat rules
Goal: Implement weapon selection
Log: session-20250101.md
```
